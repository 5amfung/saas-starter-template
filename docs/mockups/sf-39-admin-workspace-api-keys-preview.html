Generate new key

Admin creates a new workspace-owned API key. In the actual product flow, this would create a hard-scoped Better Auth key under `configId: system-managed` for this workspace.

Suggested post-create behavior for the real app: reveal the raw key once, then return to the list with only masked fragments visible.
